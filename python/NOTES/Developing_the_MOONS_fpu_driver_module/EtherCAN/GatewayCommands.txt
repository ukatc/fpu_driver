====== GatewayCommands ======
Created Thursday 31 October 2019

==== Special EtherCAN gateway commands ====


These are :

* delay messages, which might avoid overload of specific FPUs
* messages that configure SYNC commands, which are sent synchronously to all FPUs
* retrieval of diagnostic counters

Examples for these messages are contained in the folder "python/doc/sample-code".
